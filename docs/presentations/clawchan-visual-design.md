# Clawchan Intelligence Agency - Visual Design Scheme

## 1. Design Philosophy
### Overall Visual Direction
- CIA/Pentagon command center aesthetic - dark, tactical, high-tech intelligence interface
- Classified document feel with red "TOP SECRET" accents
- Military-grade precision with modern tech sophistication
- Palantir-inspired data visualization with dark theme

### Content Density
- High information density - intelligence dashboards require maximum data visibility
- Strategic use of negative space for emphasis on critical alerts

## 2. Color System
**Primary Palette**
- Background: #0a0a0f (Deep black - command center dark)
- Primary: #ef4444 (Alert red - classified/urgent)
- Secondary: #1a1a24 (Panel gray - intelligence console)
- Accent: #f59e0b (Gold amber - high-value intel)

**Text Colors**
- Dark text (for light backgrounds): #1a1a24
- Light text (for dark backgrounds): #ffffff with varying opacity (90%, 70%, 50%, 30%)

**Semantic Colors**
- Critical: #dc2626 (DEFCON 1)
- Warning: #f59e0b (Elevated threat)
- Success: #10b981 (Operational)
- Info: #3b82f6 (Monitoring)

## 3. Typography Design
**Heading Fonts**
- Chinese Font
  - Noto Sans SC (Modern sans-serif, military precision)
- English Font
  - Inter (Clean, technical, intelligence reports)
  - JetBrains Mono (Monospace for data/codes)

**Body Fonts**
- Chinese Font
  - Noto Sans SC (Consistent with headings)
- English Font
  - Inter (Readable at small sizes for data)
  - JetBrains Mono (For API endpoints, coordinates)

## 4. Image Style
- Dark satellite imagery aesthetic
- HUD/ heads-up display overlays
- Wireframe globe visualizations
- Redacted document styling
- Classified stamp graphics
- Night vision green accents (optional)
